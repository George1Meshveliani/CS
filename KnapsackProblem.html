Knapsack Problem: Dynamic Programming

The Knapsack Problem

The Knapsack Problem is one of the most famous problems in combinatorial optimization. Given a set of items, each with a weight and a value, determine the number of each item to include in a collection so that the total weight is less than or equal to a given limit and the total value is as large as possible.

1. Problem Statement

0/1 Knapsack Problem

Given:

  • n items, each with weight w[i] and value v[i]
  • A knapsack with capacity W

Find: Maximum value that can be obtained without exceeding the weight limit.

Constraint: Each item can be taken at most once (0 or 1).

Example

Items:

  • Item 1: weight=2, value=10
  • Item 2: weight=3, value=15
  • Item 3: weight=5, value=20
  • Item 4: weight=7, value=25

Capacity: W = 10

Optimal Solution: Items 2 and 3 (weight=8, value=35)

2. Variants

  • 0/1 Knapsack: Each item can be taken at most once
  • Unbounded Knapsack: Each item can be taken multiple times
  • Fractional Knapsack: Items can be taken in fractions (solved by greedy)
  • Multiple Knapsacks: Multiple knapsacks with different capacities

3. Dynamic Programming Approach

Recurrence Relation

Let dp[i][w] = maximum value using first i items with weight limit w

Base Cases:

dp[0][w] = 0 for all w (no items)

dp[i][0] = 0 for all i (no capacity)

Recurrence:

dp[i][w] = max(

dp[i-1][w], // Don't take item i

dp[i-1][w - weight[i]] + value[i] // Take item i

)

if w >= weight[i], else dp[i][w] = dp[i-1][w]

Intuition

For each item, we have two choices:

  1. Don't take it: Value remains dp[i-1][w]
  2. Take it: Value = value[i] + dp[i-1][w - weight[i]]

We choose the option that gives maximum value.

5. Implementation

Complete C++ Implementation

#include <iostream> #include <vector> #include <algorithm> using namespace std; int knapsack01(vector<int>& weights, vector<int>& values, int capacity) { int n = weights.size(); // dp[i][w] = max value using first i items with capacity w vector<vector<int>> dp(n + 1, vector<int>(capacity + 1, 0)); // Fill DP table for (int i = 1; i <= n; i++) { for (int w = 1; w <= capacity; w++) { // Don't take item i dp[i][w] = dp[i-1][w]; // Take item i (if weight allows) if (weights[i-1] <= w) { dp[i][w] = max(dp[i][w], dp[i-1][w - weights[i-1]] + values[i-1]); } } } return dp[n][capacity]; } // Reconstruct solution (which items were taken) vector<int> reconstructSolution(vector<int>& weights, vector<int>& values, int capacity, vector<vector<int>>& dp) { vector<int> solution; int i = weights.size(); int w = capacity; while (i > 0 && w > 0) { if (dp[i][w] != dp[i-1][w]) { // Item i was taken solution.push_back(i-1); w -= weights[i-1]; } i--; } reverse(solution.begin(), solution.end()); return solution; } int main() { vector<int> weights = {2, 3, 5, 7}; vector<int> values = {10, 15, 20, 25}; int capacity = 10; int maxValue = knapsack01(weights, values, capacity); cout << "Maximum value: " << maxValue << endl; // Reconstruct solution int n = weights.size(); vector<vector<int>> dp(n + 1, vector<int>(capacity + 1, 0)); for (int i = 1; i <= n; i++) { for (int w = 1; w <= capacity; w++) { dp[i][w] = dp[i-1][w]; if (weights[i-1] <= w) { dp[i][w] = max(dp[i][w], dp[i-1][w - weights[i-1]] + values[i-1]); } } } vector<int> solution = reconstructSolution(weights, values, capacity, dp); cout << "Items taken: "; for (int idx : solution) { cout << "Item " << (idx+1) << " "; } cout << endl; return 0; }

6. Space Optimization

We can optimize space from O(n×W) to O(W) by using only one array:

int knapsack01Optimized(vector<int>& weights, vector<int>& values, int capacity) { int n = weights.size(); vector<int> dp(capacity + 1, 0); for (int i = 0; i < n; i++) { // Iterate backwards to avoid using updated values for (int w = capacity; w >= weights[i]; w--) { dp[w] = max(dp[w], dp[w - weights[i]] + values[i]); } } return dp[capacity]; }

7. Complexity Analysis

Aspect Complexity
Time O(n × W)
Space (2D) O(n × W)
Space (Optimized) O(W)

Summary

The Knapsack Problem demonstrates:

  1. Optimal substructure: optimal solution contains optimal subproblem solutions
  2. DP table construction: building solution bottom-up
  3. Space optimization: reducing from 2D to 1D array
  4. Solution reconstruction: backtracking to find items taken
